ABOUT RESUME PROJECTS
Here's some projects that I have built for fun-
Trakt Scrobbler

Trakt Scrobbler is a Python application that automatically recognizes the currently playing media on your PC and syncs that information with Trakt servers. It features a multi-threaded model with support for sourcing information from multiple media players.

Impartus Scraper

Python application to download videos from Impartus Lecture Capture servers hosted in BITS Hyderabad. It internally uses ffmpeg to download HLS streams of the videos.

Batchsnap Sorter

The Batchsnap Sorter identifies unique faces from a corpus of photos and allows the user to retrieve images containing their face. It works by first running a face detection network on the images, extracting their feature embeddings, and finally clustering the faces using HDBSCAN.
