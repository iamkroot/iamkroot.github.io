The `<datalist>` element in HTML provides a nice "autocomplete" feature for `<input>` elements. You can specify a list of predefined options for the user.

Source: Found in the HTML source of the [CS 340 Introduction to Computer Systems](https://courses.grainger.illinois.edu/cs340/fa2024/) website at UIUC.
