The debugging feature in Python's f-strings, like `{my_variable =}`, is sensitive to whitespace. The spaces around the variable name matter.

Source: Noticed in a PyTorch tutorial - [How to use CutMix and MixUp](https://pytorch.org/vision/stable/auto_examples/transforms/plot_cutmix_mixup.html#after-the-dataloader).
